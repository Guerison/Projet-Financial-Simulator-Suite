💰 Financial Simulator Suite

Un projet Python en ligne de commande qui simule plusieurs scénarios d’analyse financière.

Ce projet a été créé pour démontrer des compétences en modélisation financière, analyse d’investissement et programmation Python dans le cadre d’un portfolio (finance / audit / conseil).

🎯 Présentation du projet

Cette application permet de simuler trois cas financiers concrets :

📈 ROI d’un investissement
🏠 Rentabilité d’un investissement immobilier
🚀 Valorisation d’une startup

Tout fonctionne en ligne de commande : simple, clair et professionnel.

🧠 Pourquoi ce projet ?

Réalisé dans le cadre d’un portfolio d’école de commerce pour mettre en avant :

Le raisonnement financier
Une logique de modélisation structurée
Une architecture de projet Python propre
Des bases de tests automatisés

Idéal pour un portfolio GitHub orienté finance, conseil, audit ou data.

✨ Fonctionnalités
📈 Calculateur de ROI

Calcule le profit et le retour sur investissement.

Formule

ROI = ((Valeur finale − Investissement initial) / Investissement initial) × 100
🏠 Simulateur d’investissement immobilier

Estime la rentabilité locative annuelle.

Formules

Revenu net = Loyer annuel − Coûts annuels  
Rendement locatif = (Revenu net / Prix du bien) × 100
🚀 Modèle de valorisation de startup

Estime la valorisation d’une startup via un multiple de revenus.

Formules

Multiple ajusté = Multiple de base + (Taux de croissance × 10)  
Valorisation = Revenu annuel × Multiple ajusté
🏗 Structure du projet
financial-simulator/
│
├── main.py
├── roi_calculator.py
├── real_estate.py
├── startup_valuation.py
├── requirements.txt
├── visualization.html
│
├── docs/
│   └── example_results.md
│
└── tests/
    └── test_financial_simulator.py
▶️ Lancer l’application

Depuis le dossier du projet :

python main.py

Puis choisir une option dans le menu :

1 - Calculateur de ROI  
2 - Investissement immobilier  
3 - Valorisation de startup  
4 - Quitter
🧪 Lancer les tests

Le projet inclut une suite de tests unitaires.

python -m unittest discover tests

Les tests vérifient les formules principales et les contrôles d’entrée.

🌐 Aperçu visuel

Ouvrir visualization.html dans un navigateur pour tester une version visuelle simple des simulateurs.

Aucune installation nécessaire.

🛠 Technologies utilisées
Python 3
Interface en ligne de commande (CLI)
Tests unitaires (unittest)
