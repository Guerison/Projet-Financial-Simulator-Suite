Financial Simulator Suite
A clean Python command-line project that simulates three common financial analysis scenarios:

ROI of an investment
Real estate rental profitability
Startup valuation using a revenue multiple
This project is designed for a GitHub portfolio and can be used to demonstrate financial reasoning, basic Python programming and structured modelling logic.

    `-- test_financial_simulator.py
Features
ROI Calculator
Calculates profit and return on investment.

Formula:

ROI = ((Final Value - Initial Investment) / Initial Investment) * 100
Real Estate Investment
Calculates net yearly income and net rental yield.

Formula:

Net Income = Annual Rent - Annual Costs
Rental Yield = (Net Income / Property Price) * 100
Startup Valuation
Estimates valuation with a simple revenue multiple approach.

Formula:

Adjusted Multiple = Base Multiple + (Growth Rate * 10)
Valuation = Annual Revenue * Adjusted Multiple
How to Run
From the project folder:

python main.py
Then choose one of the menu options:

1 - ROI Calculator
2 - Real Estate Investment
3 - Startup Valuation
4 - Exit
How to Run Tests
From the project folder:

python -m unittest discover tests
The test suite validates the main formulas and input checks.

Visual Preview
Open visualization.html in a browser to interact with a simple visual version of the three simulators.

No installation is required.

Financial Simulator Suite

Un projet Python en ligne de commande, propre et structuré, qui simule trois scénarios courants d’analyse financière :

ROI d’un investissement
Rentabilité d’un bien immobilier locatif

Ce projet est conçu pour un portfolio GitHub et peut être utilisé pour démontrer le raisonnement financier, les bases de Python et une logique de modélisation structurée.

Fonctionnalités détaillées
Calculateur de ROI

Calcule le profit et le retour sur investissement.

Formule :

ROI = ((Valeur finale - Investissement initial) / Investissement initial) * 100
Investissement immobilier

Calcule le revenu net annuel et le rendement locatif net.

Formules :

Revenu net = Loyer annuel - Coûts annuels
Rendement locatif = (Revenu net / Prix du bien) * 100
Valorisation de startup

Estime la valorisation avec une approche simple basée sur un multiple de revenus.

Formules :

Multiple ajusté = Multiple de base + (Taux de croissance * 10)
Valorisation = Revenu annuel * Multiple ajusté
